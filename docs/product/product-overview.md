# 产品概述

## 产品定义

Yannis's Blog 是一个基于 Hexo + Chic 主题的个人技术博客，记录 2019-2020 年间的技术学习笔记和成长历程。站点为纯静态站点，部署在 GitHub Pages 上。

## 核心能力

| 能力 | 说明 |
|------|------|
| 技术笔记记录与展示 | C 语言、Java、Python、高等数学等学习笔记 |
| 文章分类与归档 | 按分类、标签、时间线浏览，支持分页 |
| 技术内容渲染 | MathJax 公式、代码高亮、tocbot 文章目录 |
| 读者互动与统计 | Gitalk 评论、不蒜子访客统计、社交链接 |

## 核心问题

学习笔记分散在各处，缺乏统一的整理和展示平台。本博客将零散的学习记录集中到一个结构化的站点中。

## 产品边界

- 不做多用户内容发布平台
- 不做商业化变现功能
- 不做动态后端服务
- 不做非技术内容
