# 设计文档

## 主题设计

本站使用 [hexo-theme-Chic](https://github.com/Siricee/hexo-theme-Chic) 主题，风格简洁优雅。

### 配色方案

- 亮色模式：白色背景 + 深色文字
- 暗色模式：深色背景 + 浅色文字
- 强调色：蓝色（链接、交互元素）

### 布局

- 首页：居中头像 + 简介 + 社交链接
- 文章页：左侧内容 + 右侧目录（tocbot）
- 移动端：汉堡菜单 + 单栏布局

### 字体

- 主字体：Helvetica Neue, PingFang SC, Microsoft YaHei 等无衬线字体
- 代码字体：等宽字体
- 图标：iconfont 自定义图标
